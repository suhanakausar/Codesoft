# CODSOFT Python Internship Projects

This repository contains all the tasks completed during the CODSOFT Python Programming Internship.

## Tasks Completed

###  Task 1: To-Do List Application
A Python-based application to manage daily tasks with features like add, update, delete, and file storage.

###  Task 2: Calculator
A simple calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

###  Task 3: Password Generator
A Python program that generates strong and random passwords based on user-defined length and complexity.
